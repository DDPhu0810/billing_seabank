Kính gửi anh/chị,

Mã Code gọi ra mạng di động, cố định của anh/chị là

Thời gian áp dụng Code bắt đầu từ 10/09/2020

Anh/chị vui lòng thực hiện theo hướng dẫn khi cần sử dụng:

  • Bước 1: Bấm 0 (đối với gọi trong nước), bấm 000 (đối với gọi quốc tế) rồi bấm số cần gọi.
  • Bước 2:Trên màn hình điện thoại sẽ hiển thị:

    • Enter Authorization Code

  • Bước 3: Anh/chị vui lòng nhập mã code **** và kết thúc bằng phím #.

Để được hỗ trợ thêm anh/chị vui lòng gọi 1991 hoặc IT chi nhánh, Phòng giao dịch.

Các hướng dẫn sử dụng điện thoại đã được Bộ phận IT gửi email cho anh/chị. Anh/chị cũng có thể truy cập tại đây để tải lại các hướng dẫn sử dụng.

Bộ phận IT kính chúc anh/chị một ngày làm việc hiệu quả.
